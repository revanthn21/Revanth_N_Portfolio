01
AlertMate — Wearable IoT Fall Detection System
IEEE CONECCT 2026

A wearable fall detection locket built on Arduino Nano 33 BLE Sense Rev2 with 9-DOF IMU for real-time monitoring. EMA-based adaptive thresholding classifies falls with high accuracy; multi-channel emergency alerts via BLE and Firebase deliver end-to-end latency under 3 seconds.

Accuracy 95.2% Recall 94.1% F1 95.0% n = 105 samples Latency < 3s
